# 🌦️ Weather App

A simple and responsive Weather Application built with **React.js** that fetches real-time weather information using a **free Weather API**. This project was created mainly to understand how to fetch data from APIs, manage asynchronous requests, and display dynamic content in React.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Displays current temperature
- 💧 Shows humidity
- 🌬️ Displays wind speed
- ☁️ Weather condition information
- 🌞 Dynamic weather icons for:
  - Hot / Sunny
  - Cold
  - Rainy
- 🎨 Weather icons implemented using **Font Awesome**
- 📱 Responsive and clean user interface

---

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- Font Awesome
- Free Weather API

---

## 🎯 Purpose of this Project

This project was developed as a learning exercise with the primary goal of understanding:

- Fetching data from external APIs
- Working with asynchronous JavaScript (`fetch`)
- Handling API responses
- Managing state in React
- Conditional rendering
- Component-based development
- Displaying dynamic data in the UI

This is not a large-scale application. The main focus was on learning how React communicates with APIs and how real-time data can be displayed effectively.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/weather-app.git
```

Navigate to the project folder:

```bash
cd weather-app
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
